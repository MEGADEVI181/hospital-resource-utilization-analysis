# Hospital Resource Utilization Analysis

## 📌 Project Objective
This project aims to analyze hospital resource utilization data to improve efficiency in healthcare management. It helps in understanding patient distribution, department workload, and treatment cost trends.

## 🧠 Problem Statement
Hospitals often face challenges in managing limited resources such as beds, doctors, and departments. This project analyzes historical data to identify patterns and optimize resource allocation.

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- Matplotlib

## 📊 Dataset Information
The dataset includes:
- Patient ID
- Year and Month
- Department
- Doctor Assigned
- Beds Used
- Stay Duration
- Treatment Cost

## ⚙️ How to Run
1. Download the project files
2. Install required libraries:
   pip install pandas matplotlib
3. Run the program:
   python hospital_analysis.py

## 📈 Output
- Total patients
- Average stay duration
- Department-wise analysis
- Year-wise cost trends
- Graphs for visualization

## 🔍 Inference
- ICU departments use more resources
- Some doctors handle higher workload
- Treatment cost shows variation over years
